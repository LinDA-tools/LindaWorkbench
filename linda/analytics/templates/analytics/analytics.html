Analytic Process Description:

Completing the description of the analytic process is not obligatory but very important in order to have a verbose description of what you try to do within analytics tool.

Some Question that might help you to write a meaningful description are:

  • What is the context of the datasources (queries or csv) that you wish to analyze?

    (ex. Radon Test Results By Town: Beginning 1987)

  • Where the datasources are found or how they are created?

    (The DOH Radon Program contracts with a radon testing laboratory to provide short-term charcoal radon test kits, radon test kit analysis and results to residents. The contract laboratory provides the radon test results to the individual home owner and the DOH Radon Program. All testing data is entered into our database.)

  • What are you plan to do with the results if they are meaninful to you?

    (ex.From this database, we are able to create radon prevalence maps, design special outreach activities and campaigns, and track the location in the home where the detector was placed.)

  • What analysis do you plan to realize and what are you initial hipotesis about the upcoming result?

    (ex.Run a multiple linear Regression analysis so as to calculate Randon metrics depending on other related parameters such as region, house materials etc....)

  • How can one get informed more about the analysis topic?

    (ex.For more information, check out http://www.health.ny.gov/environmental/radiological/radon/radon.htm)

Tips to have a succesful analytics process:

Keep in mind...:

  • Make sure you have played with the sample processes in order to get familiarized with the analytics workflow.

  • In case of using an algorithm that requires both training and evaluation datasets , Please be sure that both have the same fields at the same order.

  • In case of creating an analytic process that makes some kind of prediction, be sure the field to predict is the last field of your query dataset or the last column of you tabular input format.

  • In case of using a forecasting algorithm make sure dates are the first field of the input dataset/s.
